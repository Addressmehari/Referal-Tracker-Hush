🛠️ Integration Guide

Initialize User & Get Code

Call this when a user logs in to get their unique referral code.

curl -X POST "/api/users/generate-code" \
  -H "X-API-KEY: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"external_user_id": "user_123"}'

Track Successful Referral

Call this on the signup/login logic of the new user (referee).

curl -X POST "/api/referrals/track" \
  -H "X-API-KEY: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "referee_id": "new_user_456",
    "referral_code": "ABC-789"
  }'
